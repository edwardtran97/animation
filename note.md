# Transform
    - transform: translate(X, Y); Dịch chuyển vị trí của element
    - transform: scale(X, Y); Zoom to/ nhỏ so với element gốc
    - transform: rotate(...deg); Xoay element
    - transform: skew(X, Y); bóp méo so với element gốc
